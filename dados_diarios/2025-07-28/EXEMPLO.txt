## Formato dos arquivos
Os arquivos devem seguir o mesmo formato dos dados existentes:
- Separados por tabulação (TAB)
- Codificação: latin-1
- Colunas obrigatórias: Data, Cliente, Total_Venda (ou Total), etc.

1. Fazer backup dos dados atuais
2. Integrar os novos dados com os históricos
3. Gerar arquivos consolidados atualizados
4. Salvar tudo nesta mesma pasta
